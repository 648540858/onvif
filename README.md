# ONVIF
ONVIF Node.js implementation based on this [article](http://ltoscanolm.hubpages.com/hub/onvif-programming-revealed).

##Supported methods
* getSystemDateAndTime
* getCapabilities
* getVideoSources
* getActiveSources
* getProfiles
* getServices
* getDeviceInformation
* getStreamUri
* PTZRelativeMove
* PTZAbsoluteMove
